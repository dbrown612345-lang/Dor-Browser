# Dor-Browser
my own browser


#install

first download the browser.js and the open.bat and move them both to your desktop

then to install the dependencys download the install.bat file and run it (it will install python and the dependencys), once the dependencys are done installing you can delete the install.bat file.

2x click on the open.bat and it should open the browser, click on the open.bat evey time you want to open the browser.

#toubleshooting

if the browser isent lanching try these steaps:

1. make shure the open.bat and the browser.js are in the same place (directory)
2. try manulaly installing python (you can do this from their website or the microsoft store) also try installing the dependencys manualy (after youve installed python of corse), open your terminal and run: pip install PyQt6 PyQt6-WebEngine PyQt6-Multimedia yt-dlp vt-py openai requests
3. make sure you have python 3.13
